# Led

Control of the LED screen.

```cards
led.plot(0, 0);
led.unplot(0, 0);
led.point(0, 0);
led.toggle(0, 0);
led.brightness();
led.plotBrightness(0, 0, 255)
led.setBrightness(255);
led.stopAnimation();
led.plotBarGraph(0, 0);
led.setDisplayMode(DisplayMode.BlackAndWhite);
led.enable(false)
```
